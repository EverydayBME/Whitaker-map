# Whitaker Map

Interactive archival map of international host institutions affiliated with the Whitaker International Biomedical Engineering Fellowship (2005–2017).

This project documents and visually represents universities, laboratories, research institutes, and biomedical engineering centers that hosted U.S. biomedical engineering students through the Whitaker International Program.

The goal of the project is to help future biomedical engineering students identify international institutions with a demonstrated history of supporting visiting U.S. BME researchers and trainees.

## Main Files

- `index.html` — interactive website prototype
- `Sheet2_Institutions_for_Map.csv` — institution-level map dataset

## Project Background

This project extends the original *Biomedical Engineers of the World* initiative and was developed through the Whitaker International Program Concluding Initiative Grants by the Baltimore Alumni team.
